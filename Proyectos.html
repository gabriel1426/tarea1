Boleteria de estadio: Boletería de estadio fue un software diseñado para la materia de POO, y este básicamente es un sistema que consistía en apartar boletas para un partido de futbol.
Las funcionalidades de este sistema eran: permitir el registro de un partido indicando su fecha y contrincantes, vender boletas al público, este apartado tenía un sección en cual se mostraban todos los asientos disponibles y ocupados que tenía cada partido, para que de esta forma los clientes pudiesen escoger de una manera más grafica los asientos que deseaban ocupar
Batalla naval: Batalla naval es un juego desarrollado hace dos años para la materia POO II, el cual consistía básicamente en tratar de hundir a los barcos enemigos.
El juego tenía funcionalidades tales como: crear un servidor, realizar una conexión con el otro jugador, posicionar las naves, realizar ataques en el tablero enemigo, marcar en el tablero los ataque recibidos, etc.
Grand Slam: Grand Slam es un sistema que se desarrolló para la materia de Bases de datos, es un sistema que consistía en realizar apartado de partidos de tenis. Sus funcionalidades consistían en registrar los equipos, registrar los partidos especificando la fecha, el lugar y los equipos que participarían en ese partido, como es un sistema que se diseñó para que equipos de todo el mundo pudieran participar se tenían que tomar controles para realizar fases, en donde en las primeras fases de enfrentaban países vecinos y a medida que el juego iba avanzando dichos encuentros se iban diversificando.
Hay Ganga: Hay Ganga es una plataforma web diseñada para realizar compras online de productos de distintas categorías
Las funciones que se pueden realizar dentro de la plataforma consisten en: buscar productos, a la búsqueda de estos productos se le pueden realizar una serie de filtros tales como categorías, precios y relevancias, también se pueden agregar productos a un carrito de compras, se puede realizar la compra de los productos que se encuentren en el carrito de compras, etc.
